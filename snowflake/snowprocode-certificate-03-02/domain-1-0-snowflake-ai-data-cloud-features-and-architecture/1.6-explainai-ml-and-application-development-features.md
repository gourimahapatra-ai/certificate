# Snowflake Advanced Features (SnowPro Core)

---

## 🔹 Snowflake Notebooks
- Interactive development environment inside Snowflake  
- Similar to Jupyter notebooks  

**Features:**
- Write SQL and Python in one place  
- Use Snowpark for data processing  
- Built-in visualization support  
- Runs directly on Snowflake compute  

**Use Cases:**
- Data exploration  
- Data engineering  
- Machine learning workflows  

---

## 🔹 Streamlit in Snowflake
- Framework to build **data apps directly in Snowflake**  

**Features:**
- Build interactive dashboards using Python  
- No external deployment required  
- Native integration with Snowflake data  

**Use Cases:**
- Internal dashboards  
- Data applications for business users  
- Rapid prototyping  

---

## 🔹 Snowpark
- Developer framework for **writing code (Python, Java, Scala)** in Snowflake  

**Features:**
- Pushes computation to Snowflake (no data movement)  
- Uses DataFrame-style APIs  
- Supports UDFs and stored procedures  

**Benefits:**
- Improved performance (compute runs inside Snowflake)  
- Secure (data never leaves platform)  

---

## 🔹 Snowflake Cortex

AI/ML capabilities built directly into Snowflake

---

### ○ AI SQL Functions
- Pre-built AI functions accessible via SQL  

**Examples:**
- Text summarization  
- Sentiment analysis  
- Translation  

**Benefits:**
- No need for external ML models  
- Easy integration into SQL workflows  

---

### ○ Cortex Search
- Semantic search capability  

**Features:**
- Search based on meaning (not exact match)  
- Works on unstructured data  
- Uses vector embeddings  

---

### ○ Cortex Analyst
- AI-powered assistant for data analysis  

**Features:**
- Converts natural language → SQL queries  
- Helps generate insights from data  
- Improves productivity for analysts  

---

## 🔹 Snowflake ML

- Native machine learning capabilities in Snowflake  

**Features:**
- Model training and inference inside Snowflake  
- Integration with Snowpark  
- Supports Python ML libraries  

---

## 🔹 Explain AI/ML and Application Development Features

### AI/ML in Snowflake
- Built-in capabilities (Cortex, ML)  
- No data movement required  
- Supports:
  - Model training  
  - Prediction (inference)  
  - Feature engineering  

---

### Application Development in Snowflake
- End-to-end platform for building data apps  

**Components:**
- Snowpark → backend logic  
- Streamlit → frontend UI  
- Notebooks → development environment  

---

## 🔹 Key Exam Takeaways 🚨

- Snowpark = **Developer framework (code inside Snowflake)**  
- Streamlit = **Build apps/dashboards**  
- Notebooks = **Interactive development environment**  

- Cortex = **AI inside Snowflake**
  - AI SQL Functions → simple AI tasks  
  - Cortex Search → semantic search  
  - Cortex Analyst → natural language to SQL  

- Snowflake ML = **model training + inference inside Snowflake**  

- Core idea:  
  👉 **Bring compute to data (no data movement)**
