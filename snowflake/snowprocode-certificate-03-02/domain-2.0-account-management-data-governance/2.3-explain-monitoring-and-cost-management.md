# 💰 Snowflake Monitoring & Cost Management
---

## 📊 Monitoring and Cost Management
- Snowflake uses a **consumption-based pricing model** (pay per usage)
- Costs mainly come from:
  - **Compute (Virtual Warehouses)** → credits consumed
  - **Storage** → data stored
  - **Cloud services** → minimal but included

- Monitoring helps:
  - Control spending  
  - Optimize performance  
  - Avoid unexpected credit usage  

---

## 🚨 Resource Monitors

### ✔️ What are Resource Monitors?
- Tools to **track and control credit usage**
- Can be set at:
  - Account level  
  - Warehouse level  

### ✔️ Features:
- Set **credit quotas** (daily, weekly, monthly)
- Define **thresholds**:
  - 50%, 75%, 100% usage alerts  
- Trigger actions:
  - Notify users  
  - Suspend warehouse  
  - Suspend account  

### ✔️ Use Cases:
- Prevent budget overruns  
- Monitor warehouse consumption  
- Enforce cost governance  

---

## 🧮 Calculating Virtual Warehouse Credit Usage

- Credits are consumed based on:
  - **Warehouse size** (XS → 6XL)
  - **Execution time** (per second billing, 60-second minimum)

### ✔️ Example:
- Medium warehouse = **4 credits/hour**
- If used for 30 minutes → **2 credits consumed**

### ✔️ Key Points:
- Larger warehouses → faster queries but higher cost  
- Auto-suspend helps reduce idle costs  
- Auto-resume ensures availability  

---

## 🧾 ACCOUNT_USAGE Schema

- A shared schema in Snowflake for **historical usage and billing data**
- Typically has **latency (~45–90 minutes)**

### ✔️ Key Views:
- `WAREHOUSE_METERING_HISTORY` → warehouse credit usage  
- `QUERY_HISTORY` → query execution details  
- `LOGIN_HISTORY` → login activity  
- `STORAGE_USAGE` → storage consumption  
- `METERING_DAILY_HISTORY` → daily credit usage  

### ✔️ Benefits:
- Enables **cost analysis and reporting**  
- Supports dashboards and audits  
- Helps identify inefficient queries and warehouses  

---

## 🧠 Pro Tips
- Use **auto-suspend (e.g., 60 seconds)** to avoid idle costs  
- Right-size warehouses based on workload  
- Combine **Resource Monitors + ACCOUNT_USAGE** for full visibility  
- Regularly review query performance to reduce credit usage
